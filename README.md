# FIC

人脸图像压缩应用（Face Image Compression Application）

参考论文：[Scalable Facial Image Compression with Deep Feature Reconstruction](https://arxiv.org/abs/1903.05921v1)

## 项目结构

```
├── algorithm 算法训练模块
│   ├── dataset 数据集目录
│   │   ├── train 训练集
│   │   └── test 测试集
│   └── save 算法训练结果
│       ├── checkpoint
│       └── result
├── backend 后端模块
└── frontend 前端模块
```
