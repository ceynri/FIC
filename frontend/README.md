# FIC 前端模块

## 项目结构

```
└── src
    ├── components  组件
    ├── mixin       全局混合
    ├── plugins     插件引入
    ├── router      路由
    ├── service     服务层，提供接口服务
    ├── styles      通用样式
    ├── utils       工具函数
    ├── views       网页视图
    ├── App.vue     主模块
    └── main.js     入口
```

## 开发项目

安装依赖：

```sh
npm install
```

启动 vue-cli 热更新开发服务：

```sh
npm run serve
```

打开 `127.0.0.1:7211` 即可预览页面

将项目编译为静态文件：

```sh
npm run build
```
